# Web Playtester

A mod that adds a button to play a level in a copy of the [Web Browser Demon List](https://webdemonlist.org/) version of geometrydash.com  
  
Technically has a setting (off by default) that adds support for a scuffed fork of the [Web Dash](https://web-dashers.github.io/) mod that has more features but is **not currently allowed on the Web Browser Demon List**. *IT IS NOT READY FOR PUBLIC USE YET! For now you need to go to the "level search" manually and press enter, and the fork may be outdated until I update it (IF I update it).*
